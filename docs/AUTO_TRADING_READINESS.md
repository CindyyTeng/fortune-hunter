# 自動交易落地判斷

## 目前結論

個股曝險前緣候選已使用手續費、交易稅、滑價、零股／整股與 T+2 模擬，也能轉成下單意圖。長期 rolling validation 仍只有 169 筆交易，未達專案設定的 300 筆可靠度門檻；而且歷史驗證資料已被反覆研究，因此目前只能視為可執行候選，不可直接實盤。

## 長期驗證

- 完整回測：2016-07～2026-05，共 119 個月、335 筆交易，月均總資產報酬 2.8111%，最大回撤 -19.13%，Profit Factor 2.01。
- 固定驗證：2022-01～2026-05，共 53 個月、176 筆交易，月均 2.7487%，年化 35.67%，最大回撤 -16.62%，Profit Factor 1.93。
- 滾動驗證：2021-01～2026-05，共 65 個月、169 筆交易，月均 2.89%，年化 37.65%，連續最大回撤 -13.72%，Profit Factor 2.24。
- 同期 0050：月均 2.10%，年化 25.39%，最大回撤 -36.38%。

## 0050 與個股比例

- ETF 多資產輪動舊線的 0050 平均目標占比約 42%；0050＋0052 核心衛星舊線約 53.3%。
- 個股曝險前緣候選本身不持有 0050，0050 占比為 0%。
- 2021-01～2026-05 的獨立資金袖套比較顯示，0050 由 40% 降至 0% 時，月均報酬由 2.57% 提升至 2.89%；但月末最大回撤由 -10.75% 擴大至 -12.76%。
- 目前若以報酬優先，選擇個股 100%、0050 0%；若以降低短期波動優先，仍可保留部分 0050。這是風險取捨，不可宣稱兩項同時改善。

## 實盤前仍需完成

1. 累積全新期間、未參與調參的紙上交易，讓驗證交易數達到至少 300 筆。
2. 逐筆核對即時資料、訊號時間、委託時間與成交回報，確認沒有未來資料或不可能成交價。
3. 驗證部分成交、委託失敗、漲跌停、斷線、重送單與券商 API 對帳。
4. 通過上述驗收前，只允許 order intent dry-run，不接真實下單。
