# 自動交易落地判斷

## 目前結論

目前沒有策略可直接實盤。個股候選已模擬手續費、交易稅、滑價、零股／整股與 T+2，也能產生下單意圖；但 rolling validation 只有 190 筆交易，未達 300 筆門檻，而且同一段歷史 validation 已被反覆研究，只能視為研究候選。

## 長期驗證

- 完整回測：2016-07～2026-05，共 119 個月、335 筆交易，月均 2.8111%，最大回撤 -19.13%，Profit Factor 2.01。
- 固定驗證：2022-01～2026-05，共 53 個月、176 筆交易，月均 2.7487%，年化 35.67%，最大回撤 -16.62%，Profit Factor 1.93。
- 滾動驗證：2021-01～2026-05，共 65 個月、190 筆交易，純個股月均 3.25%，年化 43.60%，每日連續最大回撤 -14.68%，Profit Factor 2.33。
- 同期 0050：月均 2.10%，年化 25.39%，最大回撤 -36.38%。

## 風險配置比較

- 純個股：月均 3.25%，每日最大回撤 -14.68%。
- 90% 個股＋10% 0050：月均 3.17%，每日最大回撤 -14.66%，避震效果只有 0.02 個百分點。
- 90% 個股＋10% 訓練期戰術袖套：月均 3.00%，每日最大回撤 -13.43%。
- 戰術袖套在每段 54 個月訓練期選擇現金、0050 趨勢、正向槓桿趨勢或反向防守，固定套用下一段 validation；20% 與 30% 袖套都未通過訓練期的報酬／回撤雙重門檻。
- 沒有方案同時大幅提高目前 3.17% 月均並降低回撤，因此沒有新的實盤候選。

## 已修正問題

1. 波動倍率不得讓單檔曝險突破 `maxPositionPct = 30%`。
2. 0050 與戰術袖套改用完整交易日曆計價，不再漏掉個股空手日的大盤波動。
3. 最大回撤改用四段 validation 的連續每日資產曲線；舊報表 -13.72% 與 -14.08% 都有低估，停止使用。

## 實盤前仍需完成

1. 凍結策略，不再用 2021-01～2026-05 調參，累積至少 300 筆全新期間紙上交易。
2. 驗證即時資料、訊號時間、部分成交、委託失敗、漲跌停、斷線重送與券商 API 對帳。
3. 通過前只允許 order intent dry-run，不接真實下單。

## 2026-07-06 長期驗證更新

- 驗證期間：2018-11 至 2026-04，共 90 個月。
- 策略：三個訓練子區間都有足夠樣本才允許交易，否則持有現金。
- 月均總資產報酬：2.68%；同期 0050：1.96%。
- 年化報酬：34.84%；Profit Factor：2.48。
- 連續每日最大回撤：-16.07%；同期 0050：-36.38%。
- validation 交易：208 筆，未達 300 筆門檻。
- 結論：可執行邏輯與真實成本模型已建立，但資料覆蓋與交易樣本仍不足，不可直接實盤，也不可接真實券商下單；僅能保留 order intent dry-run。
