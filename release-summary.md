## 更新重點

- 整合中短期型態選股，加入 `W底 / 頭肩底 / 上升三角形 / 上升旗形 / 下降楔形`
- 新增賣出警告邏輯，空頭型態不只列為風險提醒，也會直接影響出場建議
- 持有週期改為兩週內為主，聚焦 5 到 10 個交易日的中短期交易
- 每檔股票改用不同的部位與止損寬度，不再統一寫死
- 前端 UI 改版，強化卡片資訊層次，並加入型態偏向與賣出警告顯示
- 即時 SSE 串流流程維持可用，可直接接 Render live server

## 技術內容

- 靜態前端持續由 GitHub Pages 提供
- 即時後端持續使用 `scripts/live-server.mjs`
- 推薦資料由 `scripts/generate-data.mjs` 重新生成
- `data/recommendations.json` 現在包含：
  - 型態偏向
  - 賣出警告
  - 兩週內交易規劃
  - 每檔不同的部位建議

## 操作定位

- 這個版本是中短期交易版本，不是長抱版
- 主要目標是找出兩週內有機會表態的候選股
- 若超過 10 個交易日沒有續攻，系統會偏向提醒先出場
