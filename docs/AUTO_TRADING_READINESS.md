# 自動交易準備度

## 目前結論

曝險前緣候選已使用手續費、交易稅、滑價、零股／整股與 T+2 模擬，交易規則可轉成下單意圖；但較長樣本外只有 173 筆交易，尚未達既定的 300 筆門檻，因此目前只可視為可執行候選，不可直接實盤。

## 候選績效

- 完整評估：2016-07～2026-05，共 119 個月、327 筆，月均 2.6298%，最大回撤 -19.46%，Profit Factor 2.02。
- 長樣本外：2022-01～2026-05，共 53 個月、173 筆，月均 2.6553%，年化 33.77%，最大回撤 -16.01%，Profit Factor 1.98。
- 同期 0050：月均 2.38%，年化 27.18%，最大回撤 -36.38%。

## 實盤前缺口

1. 以未參與本次選擇的新期間累積至少 300 筆樣本外或紙上交易。
2. 將每日候選資料改成可重現的 point-in-time 生產管線，確認沒有倖存者偏差。
3. 紙上交易驗證委託失敗、部分成交、漲跌停、斷線與實際滑價。
4. 通過後才接券商 API；第一階段仍需人工覆核，不得直接全自動實盤。
