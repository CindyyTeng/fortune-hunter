# 自動交易落地判斷

## 目前結論

個股候選已使用手續費、交易稅、滑價、零股／整股與 T+2 模擬，也能轉成下單意圖。長期 rolling validation 仍只有 190 筆交易，未達專案設定的 300 筆可靠度門檻；而且歷史驗證資料已被反覆研究，因此目前只能視為可執行候選，不可直接實盤。

## 長期驗證

- 完整回測：2016-07～2026-05，共 119 個月、335 筆交易，月均總資產報酬 2.8111%，最大回撤 -19.13%，Profit Factor 2.01。
- 固定驗證：2022-01～2026-05，共 53 個月、176 筆交易，月均 2.7487%，年化 35.67%，最大回撤 -16.62%，Profit Factor 1.93。
- 滾動驗證：2021-01～2026-05，共 65 個月、190 筆交易，純個股月均 3.25%，年化 43.60%，每日連續最大回撤 -14.68%，Profit Factor 2.33。
- 同期 0050：月均 2.10%，年化 25.39%，最大回撤 -36.38%。

## 目前資金比例

- 修正後確認舊程式的波動倍率可能讓單檔曝險超過 `maxPositionPct = 30%`；現在最終部位已強制鎖回 30%。
- 純個股為月均 3.25%、最大回撤 -14.68%。
- 90% 個股＋10% 0050 為月均 3.17%、年化 42.70%、最大回撤 -14.08%。
- 目前採用 90% 個股＋10% 0050：相較純個股犧牲 0.08 個百分點月均報酬，換取 0.60 個百分點回撤改善。
- 舊報表的 -13.72% 只取最差單一 fold 與月末串接，未涵蓋跨 fold 的每日高低點；已停止使用該低估數字。

## 實盤前仍需完成

1. 累積全新期間、未參與調參的紙上交易，讓驗證交易數達到至少 300 筆。
2. 逐筆核對即時資料、訊號時間、委託時間與成交回報，確認沒有未來資料或不可能成交價。
3. 驗證部分成交、委託失敗、漲跌停、斷線、重送單與券商 API 對帳。
4. 通過上述驗收前，只允許 order intent dry-run，不接真實下單。
